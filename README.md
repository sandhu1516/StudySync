# 🎓 StudySync – AI-Powered Student Planner Dashboard

StudySync is a full-stack AI-powered student productivity platform designed to help students organize, manage, and improve their academic workflow. The platform combines planning, productivity, and AI-assisted learning tools into a single modern dashboard.

## 🚀 Features

### 📋 Task Management

* Create, update, and delete tasks
* Mark tasks as completed
* Track pending and completed work

### 📅 Timetable Planner

* Manage study schedules efficiently
* Organize subjects and study sessions

### 🎯 Goal Tracking

* Set academic goals
* Monitor progress toward objectives

### 📝 Smart Notes

* Create and manage study notes
* Keep important learning material organized

### 🧠 AI Flashcard Generator

* Generate study flashcards using AI
* Subject-based revision support
* Quick concept recall and practice

### ⏱️ Pomodoro Study Timer

* Boost productivity with focused study sessions
* Track study hours automatically

### 📊 Analytics Dashboard

* Visualize learning progress
* Monitor productivity and task completion rates

### 🏆 Achievement Badges

* Earn badges based on productivity milestones
* Encourage consistent learning habits

### 🔥 Streak Tracking

* Maintain daily study consistency
* Build productive study routines

### 🔐 Secure Authentication

* JWT-based authentication
* Password hashing with bcrypt
* Protected routes and secure user sessions

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/StudySync.git
cd StudySync
```

### Frontend Setup

```bash
cd student-planner-dashboard
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Environment Variables

Create a `.env` file inside the backend folder and configure:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
COHERE_API_KEY=your_api_key
```
